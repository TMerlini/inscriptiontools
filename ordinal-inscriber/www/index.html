Ordinal Inscriber

Welcome to your Bitcoin Ordinals Inscription Tool!

This application allows you to create and manage Bitcoin Ordinals inscriptions directly from your Umbrel node.

Connecting to the Ordinal Inscriber service...

Features

  • Create text, image, and other file type inscriptions
  • Connect directly to your Bitcoin and Ordinals nodes
  • Manage your existing inscriptions
  • View transaction details and status

Requirements

This app requires the following Umbrel apps to be running:

  • Bitcoin Core
  • Ordinals

Ensure both services are running properly for the Ordinal Inscriber to work correctly.
